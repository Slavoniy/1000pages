# Генератор Сайта (1000+ страниц)

Этот проект генерирует статический сайт услуг электрики и запускает его в Docker контейнере с Nginx.

## Быстрый запуск (рекомендуется)

Если у вас установлен **Docker Compose**, просто запустите:

```bash
docker-compose up -d --build
```

Сайт будет доступен по адресу: http://localhost:8088

## Запуск вручную через Docker

Если вы хотите запустить через `docker run`, **важно правильно указать порты**. Внутри контейнера Nginx слушает порт **80**, а не 3000.

1. Сборка образа:
   ```bash
   docker build -t site-generator .
   ```

2. Запуск контейнера (пробрасываем порт 8088 на порт 80 контейнера):
   ```bash
   docker run -d -p 8088:80 --name site-test site-generator
   ```

3. Откройте в браузере: http://localhost:8088

## Ошибка, которая у вас возникла

Вы запускали команду: `docker run ... -p 8088:3000 ...`.
Это перенаправляло запросы на порт 3000 внутри контейнера, но там никто не слушал, так как Nginx работает на порту 80.
